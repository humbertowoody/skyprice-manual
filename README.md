# SkyPrice - Manual de Usuario

En este repositorio se encuentra el manual de usuario de SkyPrice, la plataforma
para la estimación de precios de mercado de departamentos en la Ciudad de México.
Que forma parte del trabajo terminal 2024-A053 de la Escuela Superior de Cómputo
del Instituto Politécnico Nacional por Humberto Alejandro Ortega Alcocer.
